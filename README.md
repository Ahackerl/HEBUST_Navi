# HEBUST_Navi
河北科技大学校园导航
这是我的数据结构课程设计，题目就是“河北科技大学校园导航”，往届都是在控制台里面实现的，这次我用 C++ 和 Qt 实现。
地点名里面的“♥”是占位符，不知为何Qt显示奇数个中文字符串乱码，出此下策。
地点坐标数据本来是储存在sqlite数据库里面的，老师不让用数据库，于是储存在了txt中。两种读取数据的代码都有，注释了其中一种方法。
这个程序还不是很完美，如果后浪发现此项目的话，可以借鉴但不要抄袭。
